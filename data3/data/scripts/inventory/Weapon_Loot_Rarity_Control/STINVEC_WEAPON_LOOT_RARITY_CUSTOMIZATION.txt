////// TO CHANGE THE ACTIVE PRISON TIMER LEVEL:

////// 1) Remove the two // marks from the far LEFT of the word 'import' on the line of the

////// NOTES__________________________________________________________________________________
//////     - This setting will have no effect unless you have also installed the mod's optional
//////       "DataDLC39_1.pk3" Prison Addon file into the "...\Dying Light\DW_DLC39" folder.
//////
//////     - If you mistakenly enable more than one level, the level that is listed upper-most
//////       in the file will be the active level.
//////
//////     - If you mistakenly disable all levels, the mod's default 'L2: INCREASED TIME 2 (+100%)' 
//////       will be the active level.
